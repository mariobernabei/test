# Sing base

``` bash
sing init
```
serve ad _Inizializzare_ la comunicazione sing

``` bash
sing add .
```
Serve ad aggiungere tutta la cartella a alla ***stanging area***

``` bash
sing status
```
Controlla se ci sono file non traccati

``` bash
sing commit -m ""
```

Per passare la cartella dalla Stanging area alla (_cartella di github / al pc_)

``` bash
sing Push
```
Salva/inserisce le nuove modifiche alla cartella già online

``` bash
sing Pull
```
sovrascrive con le nuove modifiche già online



## Utilizzo

Facilità il lavoro in gruppo, ognuno fa le sue modifiche e le manda con il comando ***push***

### Criticità

due push danno problemi al pull se ci sono modifiche sullo stesso file
